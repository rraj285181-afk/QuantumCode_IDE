Python Introduction

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

It is used for:

  • Web development (server-side)
  • Software development
  • Mathematics and Data Science
  • System scripting

Why Python?

Python works on different platforms (Windows, Mac, Linux, etc.) and has a simple syntax similar to the English language.

Example: Hello World 
print("Hello, World!")

Python Variables

Variables are containers for storing data values. In Python, variables are created when you assign a value to it.

Creating Variables

Python has no command for declaring a variable. A variable is created the moment you first assign a value to it.

Example: Storing Variables 
x = 5
y = "John"
print(x)
print(y)

Python Operators

Operators are used to perform operations on variables and values. Python divides the operators in the following groups:

  • Arithmetic operators (+, -, *, /)
  • Assignment operators (=, +=, -=)
  • Comparison operators (==, !=, >, <)
Example: Arithmetic 
a = 10
b = 3
sum_val = a + b
print("The sum is:", sum_val)

Python If...Else

Python supports the usual logical conditions from mathematics:

  • Equals: a == b
  • Not Equals: a != b
  • Less than: a < b

These conditions can be used in several ways, most commonly in "if statements" and loops.

Example: Conditional logic 
a = 33
b = 200
if b > a:
    print("b is greater than a")

Python Loops

Python has two primitive loop commands:

  • while loops
  • for loops

The For Loop

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).

Example: Loop iteration 
fruits = ["apple", "banana", "cherry"]
for x in fruits:
    print(x)

Python Functions

A function is a block of code which only runs when it is called. You can pass data, known as parameters, into a function.

Creating a Function

In Python, a function is defined using the def keyword.

Example: Defining functions 
def my_function():
    print("Hello from inside a function!")

my_function()
