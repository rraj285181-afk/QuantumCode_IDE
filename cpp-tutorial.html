C++ Introduction

C++ is a cross-platform language that can be used to create high-performance applications.

C++ was developed by Bjarne Stroustrup, as an extension to the C language.

Why C++?

C++ is one of the world's most popular programming languages. It gives programmers a high level of control over system resources and memory.

Example: Hello World 
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!";
    return 0;
}

C++ Variables

Variables are containers for storing data values. In C++, there are different types of variables defined with keywords:

  • int - stores integers
  • double - stores floating point numbers
  • char - stores single characters
  • string - stores text strings
  • bool - stores boolean values
Example: Declaring variables 
#include <iostream>
#include <string>
using namespace std;

int main() {
    int myNum = 15;
    string myName = "John";
    cout << myNum << "\n";
    cout << myName;
    return 0;
}

C++ Operators

Operators are used to perform operations on variables and values. In the example below, we use the addition operator (+) to add together two values:

Example: Operations 
#include <iostream>
using namespace std;

int main() {
    int x = 100 + 50;
    cout << x;
    return 0;
}

C++ If...Else

Use the if statement to specify a block of C++ code to be executed if a condition is true.

Example: Conditionals 
#include <iostream>
using namespace std;

int main() {
    int x = 20;
    int y = 18;
    if (x > y) {
        cout << "x is greater than y";
    }
    return 0;
}

C++ Loops

Loops can execute a block of code as long as a specified condition is reached. Loops are handy because they save time, reduce errors, and they make code more readable.

The C++ For Loop

When you know exactly how many times you want to loop through a block of code, use the for loop.

Example: For loop 
#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 5; i++) {
        cout << i << "\n";
    }
    return 0;
}

C++ Functions

A function is a block of code which only runs when it is called. You can pass data, known as parameters, into a function. Functions are used to perform certain actions, and they are important for reusing code.

Example: Declaring functions 
#include <iostream>
using namespace std;

void myFunction() {
    cout << "I just got executed!";
}

int main() {
    myFunction();
    return 0;
}
